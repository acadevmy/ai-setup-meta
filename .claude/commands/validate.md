Validazione pre-release: verifica che tutti i file referenziati dai manifest dei template esistano nel repository.

## Istruzioni

1. Esegui lo script di validazione:
   ```bash
   bash scripts/validate-setup-urls.sh
   ```
2. Se ci sono file mancanti, elencali chiaramente e suggerisci come risolvere
3. Se tutto e' OK, conferma che la validazione e' passata
